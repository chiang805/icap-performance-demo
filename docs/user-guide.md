# 使用者指南

## 登入及角色模擬

展示版預設使用範例帳號，並同時具備 Admin、HR、Manager 與 Executive Viewer 角色。角色代表功能權限，資料範圍由 data scope 控制。

## 個人工作首頁

首頁提供今日待辦、逾期工作、即將到期、OGSM 待更新、KPI 待填報、工作日誌狀態、待審核事項與每日檢核清單。

## OGSM 與 KPI

OGSM 可用樹狀與網狀關係檢視。燈號可點擊查看原因，形成原因包含更新逾期、計畫與實際進度差異、KPI 目標差異、里程碑逾期與依賴風險。

## 工作日誌

工作日誌包含日期、員工、任務、部門工作項目、OGSM、KPI、專案、實際工作內容、成果、工時、進度、問題、支援需求與下一步。正式 KPI 數值需經確認流程後更新。

## 審核與稽核

審核流程支援送審、退回、核准，並保留修改前後內容、操作者、操作時間與修改原因。
